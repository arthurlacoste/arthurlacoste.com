## Installer

Pré-requis :
- [Ruby](https://www.ruby-lang.org/fr/)
- [Jekyll](jekyllrb.com)

Ensuite un simple...

```
jekyll serve
```

...Fera l'affaire !


## Reconstruire les CGV en PDF

Parce qu'une entreprise sans CGV, c'est comme un père noël (ou un hipster) sans barbe :

```
npm run build:cgv
```
